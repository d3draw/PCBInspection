## 6. 조명

> 조명 종류는 실제 테스트 후 확정. 아래는 테스트용 구성.

### 테스트 조명 세트

| 종류 | 용도 | 수량 | 가격대 (만원) |
|------|------|:---:|:---:|
| 돔 조명 (Dome) | 균일 확산광, 부품 유무 | 1 | 20~50 |
| 저각 링 조명 (Low-angle Ring) | 솔더 텍스처, 이물 | 1 | 15~30 |
| 동축 조명 (Coaxial) | 마킹, 패드 브릿지 | 1 | 20~40 |
| **조명 컨트롤러** | 다채널 스트로브 제어 | 1 | 30~50 |

### 테스트 계획

1. 각 조명별 단독 촬영 → 결함 검출 가능 여부 확인
2. 조합 촬영 → 최적 조명 조합 선정
3. 최종 조명 1~2종으로 확정 (비용 절감)

## 7. 이송 스테이지

| 항목 | 사양 |
|------|------|
| 타입 | 기성품 리니어 스테이지 |
| 축 | X축 1축 (보드 4포지션 이동) |
| 행정 | ≥ 400 mm |
| 위치 정밀도 | ± 0.05 mm |
| 반복 정밀도 | ± 0.02 mm |
| 이동 속도 | ≥ 200 mm/s |
| 구동 | 볼스크류 + 서보 or 스테핑 |

### 후보

| 제조사 | 비고 | 가격대 (만원) |
|--------|------|:---:|
| Misumi | LX 시리즈, 범용 | 50~100 |
| THK | KR 시리즈, 고정밀 | 100~150 |
| Hiwin | 국내 유통, 가성비 | 80~120 |

### 동작 시퀀스

```
Position 1 → 촬영 → 이동(0.3s) → Position 2 → 촬영 → ... → Position 4 → 촬영
총 이동 시간: 0.3s × 3회 = ~0.9초
총 촬영 시간: 0.5s × 4회 = ~2.0초
합계: ~2.9초
```

## 11. 리스크 및 대안

| 리스크 | 영향 | 대안 |
|--------|------|------|
| 25MP GigE 전송 대역폭 (5fps 시 ~250MB/s) | GigE 1Gbps 초과 가능 | 10GigE 카메라 또는 ROI 크롭 전송 |
| Jetson 메모리 부족 (ROI 다수) | OOM | ROI 배치 분할 처리 |
| 조명 테스트 후 다종 조명 필요 | 촬영 횟수 증가 | 택트 재계산, 스트로브 전환 |
| 20μm 분해능 부족 (미세 결함) | 검출 누락 | 렌즈 교체 (고배율) 또는 2단계 촬영 |

> GigE 대역폭 이슈: 25MP × 8bit × 5fps = 125MB/s. GigE Vision (1Gbps ≈ 120MB/s) 경계. 트리거 모드 사용 시 연속 전송 아니므로 대부분 문제 없음. 연속 촬영 필요 시 10GigE 또는 USB3 검토.
